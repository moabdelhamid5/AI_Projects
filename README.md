# AI_Projects
## I'm Mo Abdelhamid, a joint MBA/MS in Applied Data Science candidate at the University of Chicago (Booth + Data Science Institute), graduating June 2026. My background spans product management, technical program management, and customer-facing AI roles at companies including Nike and Amazon Web Services, and I'm currently focused on AI product roles where I can bridge technical depth with business impact. This repository contains projects from my graduate coursework and independent work in applied AI, including a retrieval-augmented generation (RAG) chatbot built with LangChain, OpenAI, and Chroma, a multimodal search chatbot, and an agentic AI project. Across these projects I have worked hands-on with large language models, embedding models, vector databases, and multi-step AI pipelines, and I have tried to document each one in a way that reflects how I think about building AI systems: with attention to design decisions, failure modes, and real-world constraints.
